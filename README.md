## 小提示🥳

### 没有源怎么办?

`yoyo` 是支持 `zy-player` 源的, 可去下载源之后导入到 `yoyo` 中

### 怎么样获得最好的使用体验?

`macOS` 推荐使用 `iina` 来播放视频

`iOS` 推荐使用内置浏览器来播放视频
