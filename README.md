## YY播放器

使用 `Flutter` 构建, 支持 `android` | `windows` | `macos`

支持自定义源

内建了两个源, 你也可以自行添加源

官方源: https://github.com/waifu-project/assets/blob/master/yy.json

[制作源教程](docs/源制作.md)

> `iOS` 下载 `ipa` 之后, 请使用 [sideloadly](https://sideloadly.io/) 签名安装
